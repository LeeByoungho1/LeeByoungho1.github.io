About

무궁무진한 아이디어를 구현하는 과정은 정말 흥미로운 일입니다. 새로운 도전을 두려워하지 않는 프론트엔드 개발자로서, 저는 항상 새로운 기술과 도구를 배우고 적용하는 것을 즐깁니다.

'Focus on’은 특정 분야에 주력한다는 뜻을 담고 있습니다. 따라서 프론트엔드 개발자로서, 하나의 분야에 깊이 있게 주력하여 뛰어난 결과물을 만들어 회사에 실질적으로 도움이 되는 프론트엔드 개발자로 일하고 싶습니다."

Micro
Site

처음으로 소개할 프로젝트는 마이크로 사이트입니다. 이 웹사이트는 잡지를 소개하며, 마우스 드래그를 활용하여 각 챕터를 확인할 수 있는 기능을 제공합니다. 이 프로젝트에서는 완성도를 높이기 위해 lottiefiles 애니메이션을 적극적으로 활용했습니다.

Web Site
LOL

리그오브레전드(LoL) 게임 회사의 웹사이트를 React를 활용하여 개발했습니다. 각 섹션을 컴포넌트로 나누어 반응형으로 디자인했으며, 이미지 전환 시 clip-path를 활용한 애니메이션 효과를 구현했습니다. 또한, 동영상은 녹화하여 WebM 형식으로 변환하여 적용했습니다.

EventPromotion
Innisfree

첫 번째 이벤트 프로모션에서는 페이지 로드 시 나타나는 애니메이션에 주안점을 두어 사용자에게 명확하게 사이트의 취지를 전달하는 데 초점을 두었습니다. 페이지를 처음 열었을 때 나타나는 독특한 애니메이션은 사용자들에게 사이트의 주요 메시지를 뚜렷이 전달하고자 고민했습니다. 이를 통해 사용자들이 사이트의 목적을 쉽게 이해할 수 있도록 신경 쓰고, 시각적 효과를 통해 더욱 흥미를 유발할 수 있었습니다.
